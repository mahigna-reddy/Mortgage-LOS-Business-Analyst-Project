## Project Description

This project focuses on enhancing a Mortgage Loan Origination System used by borrowers, loan officers, underwriters, compliance teams, and operations managers.

The existing process had multiple manual steps, limited loan status visibility, delayed document verification, and weak audit traceability. The enhanced system allows borrowers to submit mortgage loan applications online, upload required documents, track application status, and receive automated notifications.

Internal users such as loan officers and underwriters can review applications, validate documents, update loan status, request additional information, and complete underwriting decisions. Operations managers can monitor the loan pipeline using dashboard reports.

---

## Business Problem

The current mortgage loan process is slow and manual. Borrowers do not have clear visibility into their loan application status, and internal teams rely heavily on manual follow-ups.

Key problems identified:

- Borrowers cannot easily track application progress.
- Loan officers manually review and follow up on missing information.
- Underwriters do not have a centralized view of documents and application details.
- Application status updates are delayed.
- Managers do not have real-time visibility into loan pipeline volume.
- Compliance teams need better audit tracking for status changes and decisions.

---

## Business Objectives

The main objective of this project is to improve the mortgage loan application and review process by creating a more digital, trackable, and compliant workflow.

Business objectives include:

- Enable borrowers to submit mortgage applications online.
- Allow borrowers to upload required loan documents.
- Provide real-time loan application status tracking.
- Allow loan officers to review applications and move them to underwriting.
- Allow underwriters to approve, reject, or request additional documents.
- Send automated notifications when loan status changes.
- Maintain audit history for compliance and reporting.
- Provide dashboard reports for operations managers.

## In Scope

The following features are included in this project:

- Borrower registration and login
- Online mortgage loan application submission
- Employment, income, property, and loan detail capture
- Required document upload
- Loan officer application review
- Underwriter decision workflow
- Borrower loan status tracking
- Automated email notifications
- Application audit trail
- Manager loan pipeline dashboard
- API requirements for loan application, status, document upload, and dashboard
- UAT test case preparation
- Requirement traceability matrix

---

## Out of Scope

The following items are not included in this project phase:

- Credit bureau integration
- Payment processing
- Final loan closing document generation
- Mobile application development
- AI-based underwriting decisioning
- Third-party title or appraisal vendor integration

## BA Responsibilities

As a Business Analyst, responsibilities included:

- Conducted requirement gathering sessions with loan officers, underwriters, compliance teams, and product owners.
- Created Business Requirements Document and Functional Requirements Document.
- Documented user stories with clear acceptance criteria.
- Created process flow diagrams for loan submission, document review, and underwriting workflow.
